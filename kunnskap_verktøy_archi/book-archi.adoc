WARNING: Installasjonsveiledningen mangler info om installering av _Eclipse Target Editor_. Denne behøves for å "Setting up the Target Platform"; ref.  https://github.com/archimatetool/archi/wiki/Running-and-Debugging-Archi
Tips fra https://www.eclipse.org/forums/index.php/t/1087542/: 
Use File -> Install New Software, enter or chose the URL for http://download.eclipse.org/releases/oxygen and then search for Plug-in in the text search field to choose "Eclipse plug-in development environment" as the feature you want to install.
